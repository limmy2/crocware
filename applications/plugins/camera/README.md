# Camera application for Flipper Zero

Required: Esp32-Cam module

Usage:
- Up - Increase contrast
- Down - Decrease contract
- Left - Invert colors
- Right - Toggle dithering
- Ok - Save photos to SD card (DCIM folder)
- Back - Close application
